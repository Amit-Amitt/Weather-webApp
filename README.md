# Weather Web App

A responsive weather web application built using HTML, CSS, and TypeScript.  
The project fetches real-time weather data using an external API.

## 🚀 Features
- Search weather by city
- Displays temperature and weather conditions
- Clean and responsive UI
- TypeScript for type safety

## 🛠 Tech Stack
- HTML
- CSS
- TypeScript
- JavaScript (compiled)
- Weather API

## ⚙️ How It Works
TypeScript code is compiled into JavaScript using the TypeScript compiler (tsc), and the browser executes the generated JavaScript.

## 📦 How to Run
1. Clone the repository
2. Open `index.html` in your browser
