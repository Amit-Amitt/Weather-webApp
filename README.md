# Weather Web App

A responsive weather web application built using HTML, CSS, and TypeScript.  
The project fetches real-time weather data using an external API.

## 🔑 API Key Setup

1. Go to OpenWeatherMap and create a free API key.
2. Open app.ts
3. Replace:

   const apiKey = "YOUR_API_KEY_HERE";

   with your actual API key.

4. Compile app.ts to update app.js (tsc app.ts).

## 🚀 Features
- Search weather by city
- Displays temperature and weather conditions
- Clean and responsive UI
- TypeScript for type safety

## 🛠 Tech Stack
- HTML
- CSS
- TypeScript
- JavaScript (compiled)
- Weather API

## ⚙️ How It Works
TypeScript code is compiled into JavaScript using the TypeScript compiler (tsc), and the browser executes the generated JavaScript.

## 📦 How to Run
1. Clone the repository
2. Open `index.html` in your browser
